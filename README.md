# cockroach-apoptosis-circuit

# 🪳 Synthetic Genetic Circuit: Inter-Species Biocontainment System
> **In Silico CRISPR-Cas9 Target Finder for Parthenogenetic Waste-Scavenger Control**

This project is a synthetic biology prototype designed to engineer an autonomous, eco-friendly bio-scavenger population utilizing the unique reproductive and behavioral traits of *Periplaneta americana* (American Cockroach), integrated with a fail-safe genetic kill-switch inspired by cephalopod biology.

---

## Project Concept: The Parthenogenetic Scavenger Strategy

### 1. Exponential Proliferation (Parthenogenesis)
By leveraging the **parthenogenetic (virgin birth)** capabilities of *Periplaneta americana*, a highly efficient and dense biological scavenging workforce can be generated rapidly from a single female founder without requiring mating pairs.

### 2. Autonomous Waste Remediation
These engineered bio-scavengers are deployed into contained organic waste ecosystems or landfill sites to accelerate organic matter decomposition and environmental cleanups.

### 3. Cephalopod-Inspired Biological Containment (The Kill-Switch)
To prevent ecological disruption and unmitigated population expansion, the circuit hardwires an **"Antisense Post-Oviposition Apoptotic Switch"**. Mirroring the post-spawning self-destruction trigger observed in octopuses, this synthetic logic-gate forces immediate programmed cell death (apoptosis) upon completion of egg-laying, ensuring strict generational containment.

---

## System Architecture & Dashboard Features

- **Sequence Input Area**: Accepts genomic FASTA fragments of the target regulatory regions (NCBI-sourced).
- **Circuit Logic Mode**: Dynamically simulates different operational modes (`Post-Oviposition Apoptotic Switch`, `Hyper-Omnivore Bioremediation`).
- **GC Content Analysis**: Automatically screens target fragments to optimize thermodynamic binding stability.
- **Dynamic Parameter Controls**: Fine-tune gRNA target lengths and minimum efficiency score cutoffs via an interactive sidebar.

---

## Note on Scoring Mechanism
The current prototype utilizes a bio-inspired heuristic algorithm for target efficiency scoring based on operational circuit gates. In a production environment, this pipeline is architected to seamlessly interface with deep learning models (e.g., DeepCRISPR) via automated API endpoints.
